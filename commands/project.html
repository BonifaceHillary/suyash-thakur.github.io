Nginx CDN Proxy

Optimized Nginx configuration that uses GeoIP-based routing and caching to act as a fast and efficient CDN proxy.

- Developed custom Nginx configuration using GeoIP module to route incoming traffic based on country of origin, with caching mechanism to improve performance and reduce server load.
- Provided manual configuration and Dockerfile/Docker Compose file to run CDN proxy in containerized environment.
- Defined various configuration options including proxy cache settings, cache revalidation, and serving stale content based on error types.

Javascript good first issues

Automated collection of JavaScript projects with a list of "good first issues" for newcomers to open source to contribute to

- Utilized the GitHub API to fetch repositories with the most updated good first issues.
- Wrote a script that generates a markdown file and updates the README with the list of good first issues, which is later converted to HTML and hosted on GitHub pages.
- Implemented an automated process to run the script everyday at 5:30 AM using GitHub actions.
